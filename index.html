What is LODSPeaKr?

LODSPeaKr is a framework to create Linked Data applications and publish RDF data in a fast way and without much effort.

Why another system for publishing LOD?

  • Fast and simple to install.
  • Flexible on what to show for each type of resource.
  • Creation of services based on Linked Data.
  • LODSPeaKr provides the possibility of creating your own workflow of execution of SPARQL queries in multiple SPARQL endpoints.
  • Customizable interface for humans as well as for machines.
  • Provides content in multiple serializations (RDFa, RDF/XML, Turtle, N-Triples, RDFJSON) by default.

How do I install LODSPeaKr?

In order to use LODSPeaKr, you only need Apache + PHP5 + Git (optional but highly recommended). It also makes use of ARC2 and Haanga (included in LODSPeaKr). There are several ways to install LODSPeaKr. Probably the easiest is to open a terminal and run

bash < <(curl -sL http://lodspeakr.org/install)

and follow the instructions. For more information, please go to the installation page and check the requirements if you don't get your system working after completing the installation process.
